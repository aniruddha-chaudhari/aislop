# AI Video Generator

A comprehensive Node.js HTTP server built with Express and TypeScript, featuring AI-powered automated video generation with text-to-speech capabilities using FFmpeg for professional video production. This innovative platform generates entertaining technology discussions through AI-powered conversational agents in a unique automated video production pipeline.

## What This Project Does

This project represents a cutting-edge fusion of artificial intelligence, multimedia generation, and entertainment. At its core, it creates humorous, educational conversations about technology topics through AI-powered conversational agents with distinct personality profiles, then automatically transforms these conversations into complete, professional-quality videos.

**Technical Architecture:**

Built with Node.js and TypeScript using Express.js for the backend API, this project leverages modern AI services including OpenAI's GPT models for conversation generation, Google AI for enhanced processing, and Supabase for data management. The frontend is a Next.js React application with Tailwind CSS styling, providing a responsive design that works across desktop and mobile devices.

The system uses Prisma for database management, FFmpeg for professional multimedia processing, and implements comprehensive CORS handling for seamless frontend-backend communication. Video generation utilizes sophisticated subtitle synchronization, audio processing pipelines, and automated FFmpeg-powered video composition workflows.

**Technology Stack:**

**Backend**: Node.js, TypeScript, Express.js, Prisma ORM, Supabase, OpenAI API, Google AI, FFmpeg, Multer for file handling, Zod for validation, fluent-ffmpeg for video processing, axios for API communication

**Frontend**: Next.js 15, React 19, TypeScript, Tailwind CSS, HTML5 Audio API

**AI & ML**: Google AI services, WhisperX for speech-to-text, advanced TTS synthesis, automated video production pipelines
